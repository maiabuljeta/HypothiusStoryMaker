HYPOTHIUS STORYMAKER

Use this tool to help you with the storytelling for any creative project

    Story key words:

    but then, all of a sudden, however.

    Also, don't forget, conflict on every page. Think of what can go wrong even from the simplest decisions the character makes.

    Surprise and setting up unanswered clues is a wonderful way to have fun and create very interesting things.

    Utilizing adjectives in front of every description can be a great way to add emotional impact to your story.

    Write the things that give you joy. All external feedback is opinion, and only take back the feedback you need.

    There is no such thing as bad art.

    And have fun with metaphor.
